# Intune Endpoint Security Baseline (Concise)

## Compliance Requirements
- BitLocker enabled with TPM protector  
- Secure Boot required  
- OS version within supported lifecycle  
- No high-severity Defender threats active  

## Configuration Profiles
- Attack Surface Reduction (block mode)  
- Exploit Protection enabled  
- Browser hardening  
- Credential Guard enabled  

## Security Baselines
- Microsoft Defender Baseline  
- MDM security baseline for Windows 11  

## Updates & Lifecycle
- Windows Update for Business (ring-based deployment)  
- Autopilot for provisioning  
- Local admin removal + LAPS rotation  

## Intent
Ensure all managed Windows endpoints meet modern security and compliance posture with consistent configuration.
